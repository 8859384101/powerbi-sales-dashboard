Power BI Sales Dashboard

A visually engaging and interactive Sales Dashboard built using Microsoft Power BI. This project provides insights into orders, revenue, and sales performance with a modern design aesthetic.

📂 Project Files

Sales_Dashboard.pbix → Power BI dashboard file

Orders.csv → Dataset containing order details

Details.csv → Dataset with additional sales information

Image_Sales_Dashboard.png → Preview image of the dashboard

dark-gradient.jpg → Background image used for styling

✨ Features

Interactive charts and KPIs for sales analysis

Dark gradient theme for a sleek, modern look

Drill-through capabilities for detailed insights

Visual storytelling with clean design and intuitive navigation



🚀 Getting Started

Clone this repository:

https://github.com/8859384101/powerbi-sales-dashboard.git

Open Sales_Dashboard.pbix in Power BI Desktop.

Load the datasets (Orders.csv, Details.csv) if required.

Explore the interactive dashboard.

🛠️ Requirements

Microsoft Power BI Desktop (latest version)

CSV files included in the repository

📌 Use Cases

Track sales performance across regions and categories

Identify trends and patterns in orders

Support business decision-making with data-driven insights

The dashboard leverages a dark gradient background for a professional and modern aesthetic, ensuring clarity and focus on the data visualizations.
